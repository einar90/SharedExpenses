Sharedexpenses

A simple app to track shared expenses for a couple of humans.

Download this project as a .zip file Download this project as a tar.gz file

A simpler way to track shared expenses

Most tracking apps for tracking joint expenses have a lot of features -- too many for my own liking. I wanted an app that lets me add an expense with as few taps as possible. So I made one.

Minimal configuration

The only configuration needed is entering the names of the two persons who will be sharing expenses. When this is done the names will appear on two separate buttons.

One-tap adding of expenses

When you want to add an expense you simply enter the amount paid, and tap the name of the person who paid it.

Planned features

Logo

The app will eventually have one.

Editing and deleting entries

At the moment there is no way to edit or delete existing entries; this will be added soon. In the mean time you can adjust the balance by adding fictitious purchases for the person you want to reduce the debt for.

Expense comments

The possibility add comments to entries will be added. This will probably be done by tapping the item in the log.

Ability to synchronize across two different devices

I plan to add an option to synchronize the data across two different devices, to that both people involved can add new purchases.
